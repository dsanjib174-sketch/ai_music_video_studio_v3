AI Music Video Studio V3 - Real MP4 Generator

This is a local Flask demo that creates a real MP4 music video from uploaded photos and song.
It does not use paid AI video generation. It creates a cinematic slideshow-style music video with transitions, title overlays, and audio sync.

HOW TO RUN
1. Install Python 3.10+
2. Open this folder in CMD / Terminal
3. Run: pip install -r requirements.txt
4. Run: python app.py
5. Open: http://127.0.0.1:5000

UPLOAD
- Song: MP3 or WAV
- Photos: JPG/PNG
- Select style, locations, and quality
- Click Generate Video

OUTPUT
Generated MP4 will be saved inside static/outputs.

NOTE
4K export may take time and needs a good PC.
